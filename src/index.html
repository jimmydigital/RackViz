URL Parameters (render.html)

ParamRequiredDescription
itemsyesPipe-separated device list (format below)
sizenoRack size in U, 1–60 (default 42)
titlenoHeader plaque text (default "Rack Diagram")
stylenoVisual style: futurama (default), jetsons, steampunk, or eighties. Each style bundles its own colors.
palettenofuturama, jetsons, steampunk, eighties, mono, pastel, highcontrast. Works with any style; defaults to the style's matching palette.
colorsnoComma-separated kind=hexcolor overrides, applied on top of the active palette

Item Format

label:u_height:kind[:color], joined with |. Labels may be empty and are percent-encoded (encode %, |, :, &). u_height is 1–60. color is an optional 6-char hex without # that overrides the device's color in any style.

QFX5200:1:switch|Patch%20Panel:2:patch|web01:3:server:ff0000

Styles

Pass style= to switch the look — shapes, background, and typography. Futurama (default): thick-ink retro cartoon on a dark space background. Jetsons: atomic-age pastels, rounded Googie shapes, sky background, saucer pedestal. Steampunk: brass and copper on dark wood — gears, portholes, hex bolts, and pressure gauges. '80s Neon: synthwave night sky and retro grid with glowing neon tube accents. Every style has a matching palette used by default, but any palette= can be combined with any style; per-item color overrides work everywhere. New styles are modules in src/styles/ registered in styles/index.js.

Embedding in Wikis

The diagram is generated by JavaScript, so <img> tags won't work. Embed with an iframe:

<iframe src="https://your-host/render.html?items=web01:2:server&size=4&title=My%20Rack"
        width="700" height="400" style="border:0"></iframe>

Or design your rack in the editor and copy the shareable URL. Use the editor's Download SVG button for a static file you can upload anywhere.

Converting SVG to PNG

Need a PNG (e.g., for tools that don't accept SVG)? Convert the downloaded file from the command line with rsvg-convert, which renders gradients and the '80s glow filter correctly:

# install:  macOS: brew install librsvg   |   Debian/Ubuntu: sudo apt install librsvg2-bin
rsvg-convert -o rack.png rack.svg

# 2x resolution for crisp embedding:
rsvg-convert --zoom 2 -o rack@2x.png rack.svg

Inkscape works too: inkscape rack.svg --export-type=png -o rack.png. ImageMagick's convert is not recommended — its built-in SVG renderer mangles gradients and filters.
